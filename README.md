# Lab 04 - Django Tax Calculator

A simple Django project with one app that calculates tax.

## Pages

| URL | Description |
|-----|-------------|
| `/` | Home page |
| `/<number>` | Calculates total price after 15% tax |
| `/taxrate` | Displays the tax rate |

## How to Run

```bash
pip install django
python manage.py runserver
```

Then open `http://127.0.0.1:8000/` in your browser.
